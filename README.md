# Game AI - Tugas Praktikum

Repositori ini berisi kumpulan tugas praktikum mata kuliah **Game AI / Kecerdasan Buatan** yang dikembangkan menggunakan **Unity**.

## Struktur Scene

Semua scene tersedia di `Assets/Scenes/`:

| Scene | Isi | Deskripsi |
|---|---|---|
| `SampleScene` | Tugas 1 & 2 | Implementasi dasar AI pada game |
| `AI Pathfinding` | Tugas 3 | Implementasi algoritma pathfinding untuk navigasi AI |

## Cara Membuka Proyek

1. Pastikan **Unity** sudah terinstal.
2. Clone repositori ini:
   ```bash
   git clone https://github.com/username/nama-repo.git
   ```
3. Buka proyek melalui **Unity Hub**.
4. Pilih scene yang ingin dijalankan di `Assets/Scenes/`.
